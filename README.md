# one_shot_pwn
一键部署pwn基础配置

**[-]没有针对Ubuntu20.04的优化**

### 安装选项

- 0-系统基础
- 1-换源
- 2-安装python2/3 python-pip  git vim gdb
- 3-安装pwntools+python换源
- 4-安装peda-gdb pwndbg
- 5--安装拓展工具 pwngdb
- 6-安装sublime_text_3

## 使用

```bash
git clone https://github.com/Joe1sn/one_shot_pwn
sudo chmod +x one_shot_pwn/bash.sh
sudo ./one_shot_pwn/bash.sh
```
